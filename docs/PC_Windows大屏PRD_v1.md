# 罗根智能体 · PC/Windows大屏PRD（V2 重写版）

> 范围：PC Web + Windows 桌面端（同一套大屏交互）  
> 文档目标：按“操作剧本”写清楚每步怎么点、怎么失败、怎么回退。

---

## 1. 大屏统一交互模型

## 1.1 布局
- 左栏：项目树 + Step1~Step9 导航 + 历史版本快捷入口
- 中栏：参数编辑区 + 操作按钮区
- 右栏：预览区 + 结果候选区 + 对比区

## 1.2 全局按钮
- `Ctrl/Cmd + S`：保存
- `Ctrl/Cmd + Enter`：执行当前步骤主操作
- `上一步/下一步`：步骤跳转

## 1.3 状态角标
- 步骤角标：`未开始 / 进行中 / 成功 / 失败`
- 顶栏保存态：`保存中 / 已保存 / 保存失败`

## 3. 大屏首页（项目总览）

### 3.2 字段
- `projectName`

## 4. 创作工作台（逐步骤操作）

## 4.0 通用字段
- `projectId`
- `currentStep`
- `mainVersionId`
- `isDirty`

## 4.0 通用操作

## Step4 合成语音（重点）

### 目标
在大屏下完成高效语音生产，支持系统音色与克隆音色。

### 字段定义
- `voiceMode`：system/clone
- `systemVoiceId`：系统音色ID
- `cloneAudioFile`：用户上传样本
- `speed`：0.8~1.2
- `emotion`：自然/激昂/沉稳
- `noiseReduce`：开关

### 上传样本规则
- 格式：mp3 / wav / m4a
- 大小：<= 20MB
- 时长：10s~120s
- 采样质量检查：人声清晰，背景噪声低

### 大屏操作剧本（系统音色）
1. 在中栏选择 `系统音色`
2. 选音色、语速、情感
3. 点击 `生成语音`
4. 右栏出现播放器、波形、时长
5. 点击 `采用当前语音`

### 大屏操作剧本（克隆音色）
1. 切换 `克隆音色`
2. 点击 `上传样本`，从本地选择文件
3. 上传成功后右栏显示样本信息
4. 点击 `开始模仿`
5. 成功后出现 `我的音色#编号`
6. 点击 `生成语音`
7. 试听通过后 `下一步`

### 失败分支
- 文件格式错：提示并阻止上传
- 时长不合规：提示区间要求
- 模仿失败：保留样本，允许重新模仿
- 生成失败：保持参数，允许重试

## 7. PC 与 Windows 差异约束

- 交互流程保持一致，不拆两套PRD。
- 仅实现层差异：
  - PC Web：文件选择受浏览器能力限制
  - Windows：可用本地路径与系统通知（前端层可选增强）
