## 1. 全局交互规则（手机端统一）

## 1.1 页面骨架
- 顶部：项目名 + 保存状态（已保存/保存中/保存失败）
- 中部：当前步骤内容（输入区 + 结果区）
- 底部固定按钮区：`上一步` `保存` `主操作`

## 1.2 按钮规则
- 主操作按钮文案随步骤变化（如“开始提取”“生成改写”“生成语音”）
- 未满足前置条件时主按钮置灰并显示禁用原因
- 主操作执行后进入 loading 态，按钮不可重复点击

## 1.3 状态规则（每步必须有）
- 初始态：未输入/未生成
- 加载态：处理中
- 成功态：有结果，可进入下一步
- 失败态：给出失败原因 + 重试按钮

## 1.4 通用错误码映射（前端提示）
- `NETWORK_TIMEOUT`：网络超时，请检查网络后重试
- `INVALID_INPUT`：输入格式不正确，请按要求修改
- `QUOTA_EXCEEDED`：当前操作额度不足（仅提示，不展开会员）
- `SERVER_ERROR`：服务繁忙，请稍后重试

## 3. 登录页

### 3.1 页面目标
完成登录并进入首页。

### 3.2 字段定义
- `phone`：手机号，string，必填，11位数字，正则 `^1\d{10}$`
- `code`：验证码，string，必填，6位数字
- `agreeProtocol`：协议勾选，boolean，必填，默认 false

### 3.3 按钮定义
- `获取验证码`
  - 点击条件：手机号合法
  - 点击结果：发送验证码倒计时 60s
- `登录/注册`
  - 点击条件：手机号合法 + 验证码合法 + 勾选协议
  - 成功：跳首页
  - 失败：toast 错误原因

### 3.4 状态机
- 初始：按钮置灰
- 加载：登录按钮显示“提交中...”
- 成功：路由跳转 `/home`
- 失败：保留输入内容，允许重试

## 6. 创作工作台（核心：逐步骤操作剧本）

## 6.0 通用字段（所有 Step）
- `stepIndex` 当前步骤
- `draftId` 当前草稿版本

## 6.0 通用按钮

## Step1 文案提取（已实现）

### 目标
将外部素材（链接/文本）提取成可改写文案。

### 字段
- `sourceInput`：textarea，必填，1~5000 字
- `sourceType`：select，默认“自动识别”

### 用户操作剧本
1. 用户粘贴链接或原文到 `sourceInput`
2. 点击 `开始提取`
3. 页面进入加载态（按钮文案：提取中...）
4. 成功后展示提取文本结果
5. 用户可点击 `复制结果` 或 `下一步（文案改写）`

### 失败分支
- 输入为空：按钮置灰 + 提示“请先输入内容”
- 提取失败：显示 `重试提取`

---

## Step2 文案改写（已实现）

### 目标
基于提取文案生成多个改写版本并选主版本。

### 字段
- `originCopy`：textarea，必填
- `rewriteStyle`：select（口语化/强情绪/专业）
- `rewriteLength`：select（精简/标准/扩写）

### 用户操作剧本
1. 用户选择改写风格与长度
2. 点击 `生成改写`
3. 加载完成后出现 3 个版本卡片
4. 点击某版本卡片 -> 标记为“主版本”
5. 点击 `下一步` 进入 Step3

### 失败分支
- 生成失败：提示失败原因 + `重试生成`
- 无版本被选中：`下一步` 置灰

## Step4 合成语音（重点细化）

### 目标
生成可用旁白音频；支持系统音色与“我的音色模仿”。

### 字段
- `voiceMode`：radio，`system` / `clone`，默认 `system`
- `systemVoiceId`：select（系统音色），`voiceMode=system` 时必填
- `cloneSampleFile`：file，`voiceMode=clone` 时必填
- `speed`：slider，0.8~1.2，默认 1.0
- `emotion`：select（自然/激昂/沉稳）
- `pauseOptimize`：switch，默认 true

### 上传规则（我的音色）
- 格式：`mp3 / wav / m4a`
- 文件大小：≤ 20MB
- 时长：10s~120s
- 音频要求：人声清晰、无强背景音乐

### 页面按钮
- `上传我的音频`
- `开始模仿`
- `生成语音`
- `试听`
- `重新上传`

### 用户操作剧本（系统音色）
1. 选择 `系统音色`
2. 选择音色、语速、情感
3. 点击 `生成语音`
4. 成功后展示音频播放器 + 时长
5. 点击 `试听`，确认后进入下一步

### 用户操作剧本（模仿音色）
1. 切换为 `我的音色`
2. 点击 `上传我的音频`，选文件
3. 上传成功后显示：文件名、时长、波形缩略
4. 点击 `开始模仿`
5. 模仿成功后生成 `我的音色-xxx`
6. 点击 `生成语音` 输出完整旁白
7. 试听通过 -> `下一步`

### 失败分支
- 格式错误：提示“仅支持 mp3/wav/m4a”
- 时长不足：提示“请上传 10~120 秒样本”
- 模仿失败：保留样本 + `重新模仿`
- 生成失败：`重试生成`，不清空参数
