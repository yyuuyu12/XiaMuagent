## 一、产品形态总览

| 端 | 访问方式 | 视频生成 | 目标用户 |
|---|---------|---------|---------|
| 手机 H5 | 浏览器打开链接 | ❌ | 手机用户，轻度使用 |
| PC Web | 浏览器打开链接 | ❌ | PC用户，文案为主 |
| 桌面客户端 | 下载zip，双击bat | ✅ 本地GPU | 有显卡，需要视频功能 |
| 微信小程序 | 微信内打开 | ❌ | 微信生态用户 |

**核心原则：视频生成只在桌面客户端提供，云端服务器不接触任何视频文件。**

## 四、各端职责说明

### 手机 H5（backend/public/index.html）

- 唯一文件，所有移动端改动在这里
- 功能：文案克隆、灵感发现、历史记录、个人中心
- 不做：视频生成（引导下载桌面客户端）
- 布局：max-width 480px，底部 Tab 导航

### PC Web 端（backend/public/pc.html）

- 独立文件，不和 index.html 混用
- 功能：与 H5 相同，**不含视频生成**
- 布局：左侧导航栏、双栏（左输入右结果）、支持键盘
- 视频入口：引导弹窗 → 提供桌面客户端下载链接
- API：全部调云端，与 H5 完全相同

### 桌面客户端（desktop_client/）

- 打包成 zip 分发，用户解压 + 双击 bat 即用
- 界面：`ui/index.html`（基于 pc.html 改造，API 地址指向 localhost）
- 文案功能：调云端 API（需联网）
- 视频功能：调本地服务（localhost:7860，离线可用）
- 激活：联网验证激活码，缓存 24 小时

### 云端后端（backend/routes/）

- 只处理：账号、文案AI、历史、激活码、配置
- 不处理：视频文件、音频文件、GPU推理

## 五、桌面客户端详细设计

### 启动流程

```
双击「一键启动.bat」
    ↓
检测便携 Python 环境（内置，无需用户安装）
    ↓
检测模型文件
    ├── 不存在 → 运行 download_models.py（显示进度）
    └── 存在   → 跳过
    ↓
启动 FastAPI 服务（localhost:7860）
    ↓
自动打开浏览器 http://localhost:7860
    ↓
用户输入激活码（联网验证一次，缓存24h）
```

### 本地服务接口（server.py）

```
GET  /                        提供 ui/index.html
POST /api/tts                 TTS 合成音频
POST /api/video/start         创建视频生成任务
GET  /api/video/:id           查询任务进度
GET  /api/video/:id/download  下载结果视频
GET  /api/system/gpu-info     检测GPU信息
```

### 前端 API 分流（ui/index.html）

```javascript
const CLOUD_API = 'https://你的Zeabur域名';  // 文案/账号/历史
const LOCAL_API  = 'http://localhost:7860';   // 视频生成

// 文案生成 → 云端
fetch(CLOUD_API + '/api/ai/inspire', ...)

// 视频生成 → 本地
fetch(LOCAL_API + '/api/video/start', ...)

// 激活码验证 → 云端
fetch(CLOUD_API + '/api/codes/activate', ...)
```

### 视频生成流程

```
用户操作：
  1. 输入文案（或从改写结果直接带入）
  2. 选择音频来源：
     ├── TTS合成（选音色）
     └── 本地音频文件（File Picker）
  3. 选择头像素材（本地图片文件）
  4. 点击「开始生成」

本地服务处理：
  接收请求
      ↓
  tts.py 生成音频（若选 TTS）
      ↓
  sadtalker_runner.py 推理
      ↓
  结果保存到 output/ 目录
      ↓
  返回完成，前端显示预览+下载
```

### 打包体积说明

```
便携版 Python + 所有依赖   ~400MB（压缩前）
PyTorch CUDA               ~2GB
ffmpeg.exe                 ~80MB
─────────────────────────────────
主程序包压缩后              ~1.5GB
模型文件（单独下载）         ~2GB
用户总下载量                ~3.5GB
```

## 九、技术选型

| 模块 | 技术 | 原因 |
|------|------|------|
| 云端后端 | Node.js + Express | 现有，继续 |
| 数据库 | MySQL（Zeabur托管） | 现有，继续 |
| 手机H5 | 原生JS（单文件） | 现有，继续 |
| PC Web | 原生JS（单文件） | 保持一致，无构建工具 |
| 本地服务 | Python + FastAPI | 与推理框架生态一致 |
| 视频推理 | SadTalker | 开源，本地运行 |
| TTS | 阿里云语音合成 | 已有阿里云账号 |
| 本地语音识别 | Whisper（已有） | 现有，复用 |
| 桌面打包 | 便携Python + bat | 最简单，用户门槛最低 |

## 十、不允许做的事

- 在云端后端存储或处理视频/音频文件
- 把视频生成功能加到 H5 或 PC Web
- 把 desktop_client/ 的业务逻辑写进 backend/
- 在 pc.html 和 index.html 之间共享代码（各自独立维护）
- 硬编码 API 地址、密钥、域名
- 把模型文件提交到 git
